Part 2: Implementation
    The filecpu_schedule_algo.py contains python source code that simulates CPU scheduling algorithms such as FCFS, RR, SPN, SRT, HRRN, and FB.
    A version of Python is required to run the program.

    When runnning, the user is prompted to select one of the algorithms by entering a character a-f.
    The program will then accept the name, arrival time, and processing time for each process that is desired to be added.

    If the b (RR) or f (FB) is selected, the value for time quantum will be accepted.

    After all process inforamtion is entered, the program will display each process in order in which they receive CPU time for the required time unit. 
    This is simulation for the execution of processes.
